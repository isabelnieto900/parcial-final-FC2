 Este directorio contiene el código y los artefactos para la Parte A del proyecto final de Física Computacional II, enfocada en la simulación del movimiento Browniano.\hypertarget{index_autotoc_md1}{}\doxysection{Descripción}\label{index_autotoc_md1}
Se simula el movimiento de una partícula browniana inmersa en un medio viscoso utilizando la ecuación de Langevin. La integración numérica se realiza mediante el método de Euler-\/\+Maruyama. El proyecto está estructurado con Programación Orientada a Objetos en C++.

El objetivo es observar la trayectoria característica de una partícula browniana y analizar su comportamiento difusivo.\hypertarget{index_autotoc_md2}{}\doxysubsection{Clases Principales\+:}\label{index_autotoc_md2}

\begin{DoxyItemize}
\item {\ttfamily \mbox{\hyperlink{classVector3D}{Vector3D}}}\+: Clase auxiliar para operaciones con vectores en 3D.
\item {\ttfamily \mbox{\hyperlink{classParticulaBrowniana}{Particula\+Browniana}}}\+: Representa la partícula, implementando la lógica de la ecuación de Langevin.
\item {\ttfamily \mbox{\hyperlink{classSimuladorBrowniano}{Simulador\+Browniano}}}\+: Gestiona la simulación, el paso del tiempo y el guardado de datos.
\end{DoxyItemize}\hypertarget{index_autotoc_md3}{}\doxysection{Estructura del Directorio}\label{index_autotoc_md3}
